
# Python Demo Assignment

Python assignment where you are to change the datatype of user-id from integer to string


## Run Locally

Clone the project

```bash
  git clone https://github.com/KushalHarsora/PythonDemo.git
```

Go to the project directory

```bash
  cd PythonDemo
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  python app.py
```

## Create the new get request

create a new patch request with the path /patch with the requirements mentioned in the code file. Once done submit the code by creating a fork of repo and creating a new pull request.
